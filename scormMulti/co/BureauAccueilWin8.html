Utiliser le bureau et l'écran d'accueil

Avec Windows 8.1, vous pouvez démarrer à partir du Bureau que l'on connaisait déjà sur les versions antérieures ou à partir de l'écran d'accueil moderne qui indique les applications sous forme de Tuiles dynamiques. Vous pouvez utiliser l'écran tactile, la souris ou le clavier suivant ce qui vous convient le mieux en terme d'ergonomie. Vous pouvez également utiliser les périphériques que vous souhaitez, car Windows est compatible avec pratiquement tous les accessoires. Par ailleurs, Bing Smart Search vous permet d'effectuer des recherches simultanées sur votre PC, sur le web, dans le cloud et dans les applications, cette fonction "Rechercher" est disponible a plusieurs endroits sur l'interface que ce soit sur l'écran d'accueil, dans la liste des applications ou encore dans le menu pop-up sur la droite de l'écran.
