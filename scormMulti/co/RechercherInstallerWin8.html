Rechercher un fichier ou une application

apercu recherche Win8

Plusieurs méthodes sont disponibles pour accéder à votre outil de recherche :

  • Aller chercher l'icône de recherche (loupe) sur l'écran d'accueil

  • Dans le menu des charmes (apparaissant sur la droite de l'écran, quand vous amenez votre souris dans le coin à droite)

  • Dans l'écran listant toutes les applications,

  • Directement en tapant au clavier sur l'écran d'accueil

  • Le raccourcis clavier "Windows+S".
